# Domain Glossary

This glossary defines domain-specific terminology for the Apartment Management System.

| Term | Definition | Sibling Nodes / Notes |
|------|------------|-----------------------|
| Apartment | A physical rental unit within a building. | |
| Tenant | An individual leasing an apartment. | |
| RentInvoice | A monthly bill generated for rent payment. | |
| MaintenanceTicket | A request submitted by a tenant for apartment maintenance/repairs. | |
| Payment | A transaction clearing a RentInvoice. | |
