# Planning Markdown Documents (planning-mds)

This directory houses the product requirements, specifications, decisions, and evidence files for the **Apartment Management System**.

## Layout

- `BLUEPRINT.md`: Master specification snapshot.
- `domain/`: Glossary and business vocabulary.
- `features/`: Active and archived feature specifications, stories, and registry trackers.
- `architecture/`: SOLUTION-PATTERNS.md, system designs, diagrams, and architecture decisions (ADRs).
- `security/`: Threat modeling and authorization structures.
- `operations/evidence/`: Build, planning, review, and validation audit logs.
